# Simulador de frutas em Streamlit

Esta aplicação exibe uma simulação interativa de firmeza e °Brix para diferentes frutas.

## Como correr localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy na cloud

Pode publicar esta aplicação no Streamlit Community Cloud diretamente a partir deste repositório GitHub.

### Passos
1. Aceda a https://streamlit.io/cloud
2. Clique em "Deploy an app"
3. Escolha este repositório GitHub
4. Selecione o ficheiro principal: `app.py`
5. Defina a versão Python (recomendada: 3.11 ou 3.12)
6. Clique em "Deploy"

A app vai usar `requirements.txt` automaticamente.
